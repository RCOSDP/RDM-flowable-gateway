# 研究データ公開ワークフロー例

このディレクトリには、研究データ公開のためのワークフロー例が含まれています。

## 参考資料

このワークフローは https://contents.nii.ac.jp/sites/default/files/2020-04/2019-1_flowchart.pdf に記載されている「出版社により公開が義務化されたデータの場合のワークフロー」を参考にしています。

## ワークフローファイル

### 書誌情報・根拠データ公開申請

ファイル: `書誌情報・根拠データ公開申請.zip`

研究者が研究データ管理チェックリストと公開可否判断チェックリストに入力し、管理者が承認または再検討を指示するワークフローです。

**プロセスフロー:**

1. **研究者: チェックリスト入力** - 研究データ管理チェックリスト（11項目）と公開可否判断チェックリスト（15項目）を入力
2. **システム: 管理者への通知** - 管理者に承認依頼の通知を送信
3. **管理者: 承認・再検討判断** - 研究者が入力した全チェックリスト項目を確認し、承認または再検討を選択
4. **分岐:** 承認の場合は終了、再検討の場合は研究者に通知してチェックリスト入力タスクへループバック

**特徴:**
- 再検討時のコメント整形にGroovyスクリプトを使用
- 研究者は再検討の指示を受けた場合、何度でも修正可能
- 管理者は研究者の入力内容を全て確認できる

## 公開可否判断チェックリスト

研究データを公開する前に、以下の項目を確認してください。

### 1. 研究関係者に係る公開不可能な事由の確認
- [ ] データの権利を誰が持つかの合意があるか？
- [ ] 公開に際してデータに権利を持つ全員の合意を得ているか？
- [ ] 共同研究において機密契約などの事項はないか？

### 2. 倫理規定の確認
- [ ] 倫理規定は研究計画書の段階で確認済みか？
- [ ] 研究計画書の時点から研究データに関する内容に変更はないか？
- [ ] 変更がある場合、変更箇所が倫理規定を満たしているか？

### 3. 分野・研究コミュニティの慣習確認
- [ ] 分野・研究コミュニティの慣習に照らし合わせて公開が妥当か？
- [ ] 注意が必要な例（文学研究のインタビュー、臨床研究データ、絶滅危惧種の生息場所など）に該当しないか？
- [ ] 該当する場合、事務局や学会誌編集者と公開可能な範囲や手法を検討したか？

### 4. 個人情報の確認
- [ ] 個人が特定できる情報を含まないか？
- [ ] 含む場合、個人が特定できない形に加工したか？（法令やガイドラインに従う）

### 5. 産業財産権の確認
- [ ] 出願中の産業財産権に係る、または商業化が想定されるデータではないか？
- [ ] 該当する場合、知的財産権の部署に相談し公開可否を検討したか？

### 6. 国家安全保障の確認
- [ ] 国家安全保障に係るデータを含まないか？
- [ ] 該当する場合、所属機関に安全保障貿易に係る輸出管理担当部署があれば相談したか？（なければ関係各所と打ち合わせの上、研究者自身が許可申請を行ったか）

## 研究データ管理チェックリスト

研究データの日常的な管理状況を確認してください。

### データ収集
- [ ] 研究データが適切に収集されているか？
- [ ] データ収集の方法やツールが研究計画に従っているか？
- [ ] データの正確性、一貫性、安全性が確保されているか？

### データ保存
- [ ] データが適切に保存されているか？
- [ ] バックアップがとられているか？
- [ ] 保存場所や保存形式が規定に従っているか？

### データセキュリティ
- [ ] データへのアクセス権限が適切に設定されているか？
- [ ] データの機密性を保つためのセキュリティ対策が講じられているか？

### メタデータ
- [ ] データのメタデータが適切に記録されているか？

### レビュー
- [ ] 定期的にデータの管理状況をレビューしたか？
- [ ] レビュー結果を記録し、関係者に報告したか？
